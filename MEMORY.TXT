# ПАМЯТЬ ПРОЕКТА FLASHBACK [UNOFFICIAL]

## Оригинальный мод
*   Оригинальный репозиторий: https://github.com/Moulberry/Flashback
*   Автор оригинала: Moulberry
*   Изначальная платформа: Fabric

## Статус порта
*   Платформа: NeoForge (неофициальный порт)
*   Автор порта: Kvass
*   Версия: 0.40.0 для Minecraft 26.1.2 (поддерживается 26.1.2 - 26.2.0)

## Изменения для модерации
*   Модерация (Modrinth/CurseForge) отклонила изначальную загрузку.
*   Требования: Указать исходники (GitHub) и лицензию.
*   **ЧИСТАЯ ВЕРСИЯ:** Кнопка Boosty была временно удалена из `MixinTitleScreen.java`, чтобы без проблем пройти первичную модерацию.
*   Чистый JAR-файл находится тут: `Flashback-0.40.0-for-MC26.1-Clean.jar`

## План действий
1. Загрузить исходный код на GitHub.
2. Использовать оформление в стиле порта TACZ (описание, ссылки на автора, указание UNOFFICIAL).
3. Пройти модерацию с чистой версией (`-Clean.jar`).
4. После одобрения модераторами — вернуть кнопку Boosty в код и выпустить апдейт.

## Попытка порта на 1.21.1
*   Папка `Flashback 1.21.1` содержит исходники, настроенные на 1.21.1 и NeoForge 21.1.233.
*   Сборка падает из-за более чем 100 ошибок, вызванных разницей в маппингах (например, `Identifier` -> `ResourceLocation`) и старыми версиями зависимостей (ModMenu, Sodium, Bobby и т.д.), которые были собраны под Java 25.
*   Порт на 1.21.1 требует серьезного ручного бэкпорта всего кода `ReplayGamePacketHandler` и поиска старых версий библиотек. Временно отложено.
